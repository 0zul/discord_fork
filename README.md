## Requirements
- Node.js 8.x

## Commands
```
gatsby develop
  Start the development server

gatsby build
  Build Discord_Fork
```

## First Setup
```
git clone [repo]
git submodule init
git submodule update
npm i -g gatsby
npm i
gatsby develop
```

### Authentication Server
[Follow me](https://github.com/prose/gatekeeper#setup-your-gatekeeper)  
Edit `gatsby-config.js` to point to the auth server.
